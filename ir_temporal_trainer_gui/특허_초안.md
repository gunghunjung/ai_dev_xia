# 발명의 명칭
ROI 기반 열화상 시계열 특징 추출 및 다중 학습전략 선택형 결함 판별 장치 및 방법

# 기술분야
본 발명은 열화상(적외선) 비디오 분석 기술에 관한 것으로서, 관심영역(ROI) 기반 시계열 특징을 생성하고 복수의 학습 전략(데이터 증강, 손실함수, 학습률 스케줄러, 검증 분할)을 선택적으로 결합하여 결함 여부를 판별하는 인공지능 학습/추론 장치, 방법 및 이를 실행하는 프로그램에 관한 것이다.

# 배경기술
종래의 결함 판별 모델은 단일 학습/검증 분할(single split), 단일 손실함수(cross entropy), 고정 학습률 또는 단순 옵티마이저 조합으로 구성되는 경우가 많다. 이러한 구성은 데이터 수가 제한된 산업 현장에서 분할 편향(split bias)에 취약하고, 경계가 모호한 시계열 구간에서 오분류를 유발하며, 학습 후반 일반화 성능 저하를 초래할 수 있다.

# 발명의 요약
본 발명은 상기 문제를 해결하기 위하여, 열화상 비디오에서 ROI를 지정하여 시계열 특징을 추출하고, 윈도우 샘플로 학습 데이터를 구성한 뒤, 학습 단계에서만 적용되는 시계열 증강과 선택형 손실함수 및 선택형 학습률 스케줄러를 적용하며, 단일 분할 또는 계층화 K-폴드(Stratified K-Fold) 검증을 선택 수행하는 결함 판별 프레임워크를 제공한다.

본 발명의 일 실시예에 따르면, 다음 효과를 얻을 수 있다.
1. 시계열 증강으로 데이터 다양성을 확보하여 강건성 향상.
2. 경계 샘플에서 손실함수 선택(label smoothing CE/focal loss)으로 분류 안정성 향상.
3. 스케줄러 적용으로 학습 후반 일반화 성능 개선.
4. 계층화 K-폴드 검증으로 분할 운 의존도 저감 및 성능 신뢰도 향상.
5. GUI 기반 선택형 파라미터 운용으로 실험 재현성 및 운용 편의성 확보.

# 도면의 간단한 설명
도 1은 본 발명의 전체 시스템 블록도이다.  
도 2는 ROI 기반 특징 추출 절차를 나타내는 흐름도이다.  
도 3은 시계열 윈도우 데이터셋 생성 과정을 나타내는 도면이다.  
도 4는 학습 단계에서 선택 가능한 증강/손실/스케줄러 분기 구조를 나타내는 도면이다.  
도 5는 단일 분할 및 계층화 K-폴드 검증 절차를 나타내는 도면이다.  
도 6은 GUI 파라미터 입력 및 학습 실행 화면 예시이다.

# 발명의 상세한 설명
## 1. 시스템 개요
본 발명의 장치는 다음 모듈을 포함한다.
1. 비디오 입력 모듈: 정상/결함 비디오를 수집.
2. ROI 설정 모듈: 사용자 상호작용으로 ROI를 지정.
3. 특징 추출 모듈: ROI 프레임에서 통계/히스토그램/변화량 특징 추출.
4. 데이터셋 생성 모듈: 시계열을 고정 길이 윈도우로 분할.
5. 학습 모듈: 1D CNN 기반 학습 수행.
6. 전략 선택 모듈: 증강, 손실함수, 스케줄러, 검증 방식 선택.
7. 추론 모듈: 신규 비디오에 대해 결함 확률 및 판정 출력.

## 2. 특징 추출
ROI 그레이스케일 영상으로부터 프레임별 특징 벡터를 생성한다. 예를 들어 아래 항목을 포함할 수 있다.
1. 평균(mean), 표준편차(std)
2. 분위값(p10, p90)
3. 프레임 간 절대차 기반 변화 에너지(diff energy)
4. 밝기 히스토그램 정규화 벡터

이로써 길이 T의 시계열 특징 행렬 X(T x D)를 얻는다.

## 3. 윈도우 데이터셋 구성
시계열 X를 윈도우 길이 L, 스트라이드 S로 분할하여 학습 샘플을 만든다. 각 샘플은 1D CNN 입력 형태(D x L)로 변환된다.

## 4. 학습 시 데이터 증강(학습 전용)
검증/추론에는 적용하지 않고 학습 샘플에만 다음 중 적어도 하나를 적용한다.
1. 가우시안 노이즈 부가
2. 시간축 랜덤 시프트(roll shift)
3. 랜덤 feature dropout(마스킹)

## 5. 손실함수 선택
학습 시 손실함수는 선택 가능하게 구성한다.
1. Label Smoothing Cross Entropy
2. Focal Loss(gamma)

클래스 불균형 완화를 위해 클래스 빈도의 역수 기반 가중치를 추가 적용할 수 있다.

## 6. 학습률 스케줄러 선택
옵티마이저(예: AdamW)와 함께 아래 스케줄러 중 하나를 선택 적용한다.
1. OneCycleLR
2. CosineAnnealingLR
3. 스케줄러 미적용

## 7. 검증 방식 선택
검증 전략은 다음 중 선택한다.
1. 단일 train/val 분할
2. 계층화 K-폴드(Stratified K-Fold)

계층화 K-폴드의 경우 fold별 최고 검증 성능을 집계하여 평균 성능을 산출할 수 있다.

## 8. GUI 기반 운용
사용자 인터페이스를 통해 아래 파라미터를 입력받아 학습 파이프라인에 반영한다.
1. 특징 추출 파라미터(fps, bins, window_len, stride)
2. 손실함수 선택(loss)
3. 스케줄러 선택(scheduler)
4. 증강 사용 여부(use_augmentation)
5. K-폴드 수(k_folds)

## 9. 구현 예
일 실시예에서, 모델은 1D CNN 백본과 선형 분류 헤드를 포함하고, 추론 시 윈도우별 확률의 평균으로 최종 결함 확률을 계산하여 임계값 기반 판정을 수행한다.

# 청구범위
## 청구항 1
열화상 비디오에서 관심영역(ROI)을 지정하고, 상기 ROI에 대해 시간축 특징 시퀀스를 생성하며, 상기 특징 시퀀스를 윈도우 단위 학습 샘플로 변환하여 결함 여부를 판별하는 인공지능 모델 학습 방법에 있어서,  
학습 단계에서 시계열 증강으로서 가우시안 노이즈, 시간축 랜덤 시프트 및 랜덤 feature dropout 중 적어도 하나를 적용하는 것을 특징으로 하는 결함 판별 방법.

## 청구항 2
청구항 1에 있어서,  
손실함수로 label smoothing이 적용된 cross entropy 또는 focal loss를 선택적으로 적용하는 것을 특징으로 하는 방법.

## 청구항 3
청구항 1 또는 청구항 2에 있어서,  
학습률 제어를 위해 OneCycleLR 또는 CosineAnnealingLR을 선택적으로 적용하는 것을 특징으로 하는 방법.

## 청구항 4
청구항 1 내지 청구항 3 중 어느 한 항에 있어서,  
검증 방식으로 단일 분할 또는 계층화 K-폴드(Stratified K-Fold)를 선택 수행하는 것을 특징으로 하는 방법.

## 청구항 5
청구항 1 내지 청구항 4 중 어느 한 항에 있어서,  
상기 증강, 손실함수, 학습률 스케줄러 및 검증 방식의 선택 파라미터를 사용자 인터페이스를 통해 입력받아 학습 모듈에 반영하는 것을 특징으로 하는 방법.

## 청구항 6
청구항 1 내지 청구항 5 중 어느 한 항의 방법을 수행하도록 명령어가 기록된 컴퓨터 판독가능 기록매체.

## 청구항 7
청구항 1 내지 청구항 5 중 어느 한 항의 방법을 수행하는 비일시적 컴퓨팅 장치.

# 초록
본 발명은 ROI 기반 열화상 시계열 특징 추출 및 다중 학습전략 선택형 결함 판별 기술에 관한 것이다. 본 발명은 열화상 비디오로부터 ROI 시계열 특징을 생성하고, 윈도우 데이터셋으로 변환한 후, 학습 시에 한정된 데이터 증강(가우시안 노이즈, 시간축 랜덤 시프트, 랜덤 feature dropout), 선택형 손실함수(label smoothing cross entropy 또는 focal loss), 선택형 학습률 스케줄러(OneCycleLR 또는 CosineAnnealingLR), 및 선택형 검증 전략(단일 분할 또는 계층화 K-폴드)을 결합하여 결함 판별 모델을 학습한다. 이에 따라 데이터 편향과 분할 의존성을 낮추고, 경계 샘플 분류 안정성과 일반화 성능을 향상시킬 수 있다.

4. 관할 특허청 양식에 맞춰 문장부호/용어(방법/장치/매체) 최종 정리할 것.
